# F1 迷你小游戏
自制 F1 主题休闲小游戏

## 游戏介绍
基于HTML和Node技术（HTML/JS/Python/Unity等）制作的F1车手二选一小游戏，包含计数轮次，最终结果等功能。

## 运行方式
1. 下载全部代码
2. 运行入口文件：f1-pick-one.html
3. 操作说明：点击你喜欢的车手头像即可
